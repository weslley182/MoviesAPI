# MoviesAPI

## On Windows:

 - Visual Studio 2022 Community
 - Sqlite3
 - SDK .Net 6
 
## Run commands:
  ```bash
  
 $ dotnet clean
 $ dotnet build
 $ dotnet ef database update
 
 ```
 
## To seed, On MoviesAPI:
 ```bash
 
 $ dotnet run seeddata
 
 ``` 
 
## Running the app:
 - https://localhost:7250/swagger/index.html
 
## Test, on MoviesAPI.Tests

 ```bash

 $ dotnet test
 or Test/Run All Tests on Visual Studio


 ```
