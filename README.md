# MoviesAPI

## On Windows:

 - Visual Studio 2022 Community
 - Sqlite3

## Run commands:
  ```bash
  
 $ dotnet clean
 $ dotnet build
 $ dotnet ef database update
 
 ```
 
## To seed:
 ```bash
 
 $ dotnet run seeddata
 
 ``` 
 
## Running the app:
 - https://localhost:7250/swagger/index.html
 
## Test

 ```bash

 $ npm run test


 ```
